A directory of open civic data resources
for the Pacific Northwest.

Learn about and build apps with civic data for the Pacific Northwest.

This project is new! Help build this directory of tools & resources!

You'll notice that not many resources are currently listed. nwdata.org started on November 7, 2013, and we'll be building out the resources with the goal of having an awesome directory by the end of the year.

This project is open source!

Contribute to nwdata.org by forking the repository on GitHub. If you're not about to fork the repository and just want to suggest a data source, app, or other resource, you can currently make that suggestion via our issues queue on GitHub.
